## 🤔 삼성 SW 역량 테스트 A형

![SW 역량테스트](https://github.com/user-attachments/assets/553d4f5e-3985-4b4d-bf9d-efb4cc8bb188)

소프트웨어 직군 지원자 대상으로 SW 문제해결 역량 측정 목적으로 도입, 주어진 문제에 대한 답안을 직접 코딩하여 문제를 해결하는 실기 방식의 테스트

<figure>
<img src="https://github.com/user-attachments/assets/7dac01c1-8fb7-40f0-863a-e69e6b605718">
<figcaption>출처: SWEA</figcaption>
</figure>


SW 역량테스트에서 취득할 수 있는 등급은 다음과 같습니다.

- A(Advanced)
- B(Professional)
- C(Expert)

예전에는 SWEA에서 상시 검정을 통해 시험 신청 후 테스트를 볼 수 있었으나, 코로나 이후 상시 검정이 무기한 연기되고, 현재는 삼성전자, 삼성 SDS 알고리즘 특강 수강생, SSAFY 교육생 등 특정 조건을 만족해야 응시 기회가 주어집니다.

📍 SSAFY 12기에서는 A형 제한 시간이 2시간으로 변경되어 진행되었습니다.

## 📌 결과

![결과](https://github.com/user-attachments/assets/054b68e6-7535-4f02-aa33-d5264f6c07ce)

### 🧐 왜 준비했는가?

> 수료 조건 및 B형 응시 자격
> 
- 1학기 수료 조건
    - 비전공자 : IM 취득
    - 전공자 : A 취득


저는 분반테스트 때 IM을 취득하였기 때문에 수료 조건은 충족하였습니다.
하지만, 알고리즘 실력도 쌓고, B형 욕심도 조금 있었기 때문에 A형은 자연스럽게 따고 싶다는 욕심이 생겼습니다.

📍 IM과 A형은 모의 시험 / B형은 삼성 코테 PASS 적용 (서류 합격 시)

A형은 두 번만에 취득하였습니다.

### ✨ 합격 기준

- A : 2문제 중 하나만 맞았을 시
- A+ : 2문제 중 두 개 다 맞았을 시


### 📃스펙

- 백준 티어: 골드3
- 언어 : Python

## 🧑🏻‍💻 공부 방법

처음 A형 응시 때는 SWEA에 있는 문제보다 백준 문제를 많이 풀어보았습니다.
백준 문제도 물론 좋지만, SWEA에 있는 문제가 A형 문제와 좀 더 흡사하였습니다. (당연하게도)

따라서, SWEA에 있는 20문제를 꼭 한 번이라도 훑고 가시는 걸 추천드립니다.

![스터디](https://github.com/user-attachments/assets/ace6f37f-26a3-484c-9dd1-e113318b5f30)

저는 스터디도 활용하여 하루에 무조건 한 문제 이상씩 A형 문제를 돌리는 것으로 진행하였고, 다른 사람들의 코드를 참고하면서 효율적인 문제 풀이 방법을 공유하는 식으로 진행하였습니다.
(추가적으로도 따로 문제를 더 풀어봄)

### 🔔 응시 팁

- dfs, bfs, 백트래킹, 시뮬레이션 문제 위주로 풀기
- 두 문제 다 읽어보고 풀이 방법이 떠오를 것 같은 문제 먼저 풀기
  (자신감 있다면 순서대로 풀어도 상관없음 / 두 문제 중 하나만 풀어도 A형을 취득하기 때문)
- 디버깅 연습 해보기
- swea로 제출 방법 익숙해지기
