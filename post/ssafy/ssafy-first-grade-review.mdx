## ✨주요 성과

1. **자격증 3개** 취득
   - 정보처리기사
   - 데이터분석 준전문가 (ADsP)
   - SQL 개발자 (SQLD)
2. **A형** 인증 취득
3. **관통 프로젝트 우수상** 수상
4. **모바일 청첩장** 토이 프로젝트
5. **이달의 동료상 2회** 수상
6. 대기업 및 공기업 **총 10개 기업 지원**
   - **삼성** **서류** 합격

## 🕰1학기로 돌아간다면?

1. 그래도, **자격증** 취득
   - 1학기 스타트캠프는 비교적 여유로움
   - 정보처리기사, SQLD는 난이도가 쉽다고 생각
2. 프로젝트 전 **직무** 설정
   - 알고리즘 공부를 시작하기 전, 본인의 희망 직무를 명확히 설정
   - **특히, 프론트엔드에 대한 개념을 잡으시는 것을 추천**
     UI/UX 디자이너와 다름. 구현 능력이 중요하며 디자인은 웹 디자이너나 퍼블리셔와 협업하는 것.
   - 본인의 흥미와 강점을 미리 파악
     예로, 본인은 디자인에 관심이 있어 자연스럽게 사용자에게 보여지는 프론트엔드 관심 ⇒
     공부하다보니 UI/UX 외에도 API 연동, 성능 최적화, 상호작용 등 이런 역할에 흥미가 생김.
3. 지원을 **망설이지 말기**
   - 경력이 부족해도 지원을 주저하지 말 것
   - 대기업 공채 자소서를 작성하며 본인의 부족한 점을 깨닫고 보완할 수 있음
   - 서류 합격은 자신감을 키워주는 중요한 경험
4. 알고리즘 공부 **균형 잡기**
   - 하루 할당량만 소화하고, 남은 시간에는 직무와 관련된 개발 공부 추천
     프론트엔드 : Javascript, React (**유데미** 추천)
     백엔드 : Java, SpringBoot (**인프런** 추천)
   - **추천 프로젝트**: **CRUD** 기능 구현 (게시판, 쇼핑몰 프로젝트)
     본인도 처음에 게시판 프로젝트를 진행하며 도움이 많이 됐음

## 💪2학기 목표

> 새해 다짐 !

1. **영어 자격증** 취득
   - 목표 기업의 요구사항에 맞게 준비
   - 토익, 오픽 등 나에게 맞는 시험을 선택하고, 꾸준히 공부하여 좋은 성과를 목표로 삼기
2. **개인 블로그**
   - 개발 관련 학습 내용을 정리하고, 프로젝트 경험을 공유하는 블로그 개설
     깃헙 블로그를 다시 제작하고 싶은 마음이 다시 듦
3. **인스타** 개설
   - 유튜브 채널 '너드 애니멀즈'의 공식 인스타그램 계정 개설
   - 유튜브 콘텐츠와 관련된 정보를 공유하고, 팔로워들과의 소통을 위해 계정 운영
4. 나만의 서비스로 **돈 벌기**
   - 웹/앱이든 상관없이 돈 한 푼이라도 벌어보기
5. 밀린 컨텐츠 바쁘지 않을 때 하나씩 업로드 해보기…😥
6. **취업 싸탈**⭐
   - 원하는 기업 꼭 싸피 2학기가 끝나기 전 싸탈!
